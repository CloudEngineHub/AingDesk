# AingDesk

[English](README.md)

AingDesk是一款简单好用的AI助手，支持知识库、模型API、分享、联网搜索、智能体，它还在飞快成长中。

## 🚀一句话简介

简单好用的AI助手软件，支持本地AI模型及API+知识库搭建。

## ✅核心功能

- 一键部署本地AI模型及主流模型API
- 本地知识库
- 智能体创建
- 可在线分享给他人使用
- 支持联网搜索
- 支持服务器端部署（即将上线）
- 单次多模型同时对话（即将上线） 

## ✨产品亮点

- 简单好用，对AI新手友好
